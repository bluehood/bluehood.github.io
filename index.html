PENETRATION TESTING

A Penetration Test is an authorised cyberattack on a computer system or network designed to test the security of a system. Learn how Security Professionals breach computer systems with practical examples in a safe home lab environment.

SHELLCODE

Shellcode is code that is used a payload in the exploitation of vulnerable software. So named as it typically gives the attacker a command shell on the vulnerable system. Get your hands dirty writing shellcode and gain a deeper understanding of binary explotation.
